Fluid columns

.col-#{i} specifies the percentage of the span width in terms of the parent container

.col-1
.col-2
.col-3
.col-6

Fixed width columns

If the parent container is shell, a single column width is 66px. The gutter between columns is 16px. 12 columns fill the shell.

Shifted columns

.shift-#{i} specifies the number of columns shifted from left. .shift-#{i}-reverse specifies the columns shifted from right.

.col-2
.col-2.shift-2
